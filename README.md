## Features
- Visual Tag Library: Click to instantly add keywords for Characters, Outfits, Lighting, and more.
- Prompt Matrix: Fine-tune specific sections of your prompt (Style, Shot, Environment).
- AI Assistant: Uses Google Gemini to expand simple ideas into detailed descriptions.
- Multi-Language: Supports English, Chinese, and Japanese.

## How to Run
https://gemini.google.com/share/86be804ef83e

## How to Edit:
Click "Try Gemini Canvas"

## Save to Your Computer:
- Save the AI_Visual_Prompt_Studio_Pro.html file to your computer.
- Double-click to open it in any modern web browser (Chrome, Safari, Edge).
- Start clicking tags to build your prompt!
- Note: To use the "Generate Keywords" button, you may need to enter your own Gemini API key depending on which version of the file you are using.
